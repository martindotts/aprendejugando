# Aprende Jugando

An educational video game that teaches programming to kids through gameplay.

[![Play Now](https://img.shields.io/badge/Play%20Now-Click%20Here-brightgreen?style=for-the-badge)](https://aprendejugando.vercel.app)

(mouse required)

## What is it?

Aprende Jugando is a game that introduces fundamental programming concepts in a fun and intuitive way. Players learn by programming directly: creating movements, defining actions, and building functions within the game itself.

## Features

- **Hands-on learning**: No confusing syntax. Kids understand programming concepts while playing.
- **Visual programming**: Movements, actions, and functions are controlled intuitively.
- **Progressive difficulty**: Starts simple, becomes increasingly challenging.

## Technology

Built with **Construct 3**, a 2D game engine based on HTML5 that requires no coding.

## Target audience

Children aged 8-14 with no prior programming experience.
